# Code of conduct

We are committed to a respectful, harassment-free project for everyone.
Contributors should be constructive, assume good intent, protect private data,
and focus criticism on code, evidence, and claims rather than people.

Unacceptable behavior includes harassment, discrimination, threats, doxxing,
publication of private vulnerability details, or sustained disruption.

Report conduct problems privately through the repository owner's GitHub profile
or the security advisory channel. Maintainers may edit or remove contributions,
temporarily restrict participation, or permanently ban participants when needed
to protect the project and its community.
